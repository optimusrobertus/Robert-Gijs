Machine Learning: Information


What is machine learning?

Machine learning is the study of algorithms by computers, in order to effectively perform a specific task. Using machine learning, computers will keep learning, and therefore will keep on improving their ability to perform that specific task.

The 5 different types of machine learning:
Type How does it work? Usecases
Supervised learning The computer receives input and output, and learns to recognize reoccuring patterns. The data received by the computer program is labeled by a machine learning enginere or data scientist.
  • Spamfilters
  • Face recognition-software
Unsupervised learning In unsupervised learning, a computer learns how to group, cluster and/or organize data in such a way that humans (or other algorithms) can make sense of it. The data the program receives is unlabeled.
    Recommender systems (like recommended YouTube videos)
Semi-supervised learning Semi-supervised learning is a combination of supervised and unsupervised learning. This means that the computer program receives both labeled and unlabeled data. This is done, because labeling is timeconsuming, and prone to human error. Thus, by adding unlabeled data, the eventual algorithm produced will be more accurate.
  • Speech recognition
  • Genetic sequencing
Reinforcement learning In this type of learning, a computer learns by analyzing its mistakes, and will improve on said mistakes. I like to think of it as evolution, as a computer program adjusts itself to its environment.
  • Robotics
  • Deep learning
  • Games

Machine learning in our daily lives

Everyday, we make use of machine learning. When you wake up, and open YouTube, to check out the latest videos in your feed, those videos have been selected using machine learning.
When you ask Siri what the weather is like today, or ask Google Assistant what's on your schedule for the day, the responses have been produced by the use of machine learning. That is because these programs need to recognize speech, questions and orders, and need to know how to properly respond.

And now, to answer what this website is all about, shoudl we be afraid of machine learning?

Having seen what machine learning is used for, I believe we should not be afraid of machine learning. Instead, we should embrace this technology, as it makes our daily lives a lot easier.
